Javascript
What is JavaScript?
JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives et à ce titre est une partie essentielle des applications web. Avec les langages HTML et CSS, JavaScript est au cœur des langages utilisés par les développeurs web.
Java & JavaScript?
Java est majoritairement utilisé côté serveur (back-end), pour gérer des bases de données, des connexions utilisateurs, etc. JavaScript est principalement utilisé côté client (front-end), pour améliorer l'expérience utilisateur.
Can i learn Javascript without knowing Java?
Of course, you can learn JavaScript without learning. Because both are different languages. Java code run in a virtual machine JVM, while JavaScript code runs on a browser. You can JavaScript through numerous ways such as youtube, materials, Books, Institutions and Online courses.
Javascript & ECMAScript
ECMAScript is a set of language rules, and JavaScript is an implementation of that standard that is compatible across servers & browsers.
